# MaJo — трекер саморазвития и персональной статистики

MaJo — это приложение для Android (в будущем также Desktop Compose), которое помогает отслеживать различные активности, строить статистику и измерять прогресс в саморазвитии.  
Проект создаётся как pet-project и учебная площадка для освоения архитектуры, Compose, Room и Clean-подходов.

---

## 🚀 Цели проекта

- Сформировать привычку регулярного саморазвития  
- Вести учёт полезных активностей (спорт, здоровье, учёба, продуктивность и т.д.)
- Упростить сбор статистики
- Визуализировать прогресс
- Использовать единую систему очков, отражающую общий уровень дисциплины

---

## 🧭 Концепция (Манифест приложения)

### Основные сущности
- **Action** — тип активности (бег, медитация, программирование)
- **ActionRecord** — запись о выполнении активности
- **PeriodSummary** — статистика за день/неделю/месяц
- **UserSettings** — настройки приложения

### Типы активностей
Поддерживаются:
- Distance (км/метры) (например: бег, ходьба, объем спринтов)
- Time (минуты/часы) (например: чтение, программирование, медитация)
- Count (количество повторений) (например: количество дневных шагов)
- Binary (сделано/не сделано) (например: вредные привычки)

### Система очков
Каждая активность имеет своё значение количества очков за единицу работы.  
Очки используются для универсальной метрики прогресса.

### MVP (первая версия)
- Список активностей
- Добавление активности
- Добавление записи
- Дневная и месячная статистика
- Простая диаграмма прогресса

## 🏗 Архитектура проекта

Проект построен на **Clean MVVM**:

```

UI (Compose)
↓ ViewModel
Domain Layer (UseCases + Models)
↓ Repository Interface
Data Layer (Room / Repository Impl)

```

### Слои:

#### **UI**
- Jetpack Compose
- StateFlows
- Экран → ViewModel → UseCase

#### **Domain**
- `model/` — чистые модели данных
- `usecase/` — бизнес-логика
- `repository/` — интерфейсы репозиториев
- `service/` — PointsCalculator и вспомогательные сервисы

#### **Data**
- Room: DAO, Entity, Database
- Реализация репозиториев

#### **DI**
- Koin (или Hilt)
